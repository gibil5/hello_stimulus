## Rutter 

### Dependencies Webpacker
 
```
webpack 
node.js 
yarn,      a replacement for the npm, to add node modules to Rails  
```

### Add to rails 
```
gem 'webpacker'
```

### Install 
```
brew install yarn

brew install node 
  sudo chown -R $USER /usr/local
  brew link node
  
rails webpacker:install 
```
